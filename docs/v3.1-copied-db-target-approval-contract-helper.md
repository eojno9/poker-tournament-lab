# v3.1 Copied DB Target Approval Contract Helper

## Purpose

This step adds a pure helper that describes whether a future copied-DB-only import rehearsal request satisfies the approval contract. It does not copy a database, read a database, write a database, run an import, connect a product route, create an API/UI flow, or add a package script.

The helper is intentionally limited to in-memory inputs and deterministic output.

## Helper

- Source: `packages/core/src/hrcCopiedDbTargetApprovalContract.ts`
- Tests: `packages/core/test/hrcCopiedDbTargetApprovalContract.test.ts`
- Export: `packages/core/src/index.ts`

The helper builds a `HrcCopiedDbTargetApprovalContract` from:

- requested operation
- approval mode
- copied DB path guard result
- copied DB write request flag
- production DB write request flag
- schema migration request flag
- route/API/UI/package exposure request flags
- raw HRC access request flag
- report JSON write request flag

## Approval Modes

Supported approval modes:

- `NONE`
- `EXPLICIT_FLAG`
- `EXPLICIT_TOKEN`

`EXPLICIT_FLAG` and `EXPLICIT_TOKEN` can make a future copied DB rehearsal eligible only when the copied DB target already passed the path guard and no forbidden scope is requested.

The helper does not store approval tokens in output.

## Decisions

The approval contract can return:

- `PREVIEW_ONLY_NO_APPROVAL_REQUIRED`
- `ELIGIBLE_COPIED_DB_REHEARSAL_APPROVAL`
- `BLOCKED_MISSING_APPROVAL`
- `BLOCKED_INVALID_APPROVAL`
- `BLOCKED_COPIED_DB_TARGET`
- `BLOCKED_PRODUCTION_DB_WRITE`
- `BLOCKED_SCHEMA_MIGRATION`
- `BLOCKED_PRODUCT_IMPORT_SURFACE`
- `BLOCKED_PACKAGE_SCRIPT`
- `BLOCKED_RAW_HRC_ACCESS`
- `BLOCKED_REPORT_JSON_WRITE`

## Safety Policy

- Production DB write remains forbidden.
- Copied DB write is not executed by this helper.
- `productionDbWriteAllowed` is always `false`.
- `copiedDbWriteAllowed` is always `false`.
- `reportFileWriteAllowed` is always `false`.
- `actualDbWritePerformed` is always `false`.
- `productImportRouteConnected` is always `false`.
- `apiUiImportFlowConnected` is always `false`.
- `packageScriptAdded` is always `false`.
- `rawHrcAccessed` is always `false`.

The helper may return `futureCopiedDbWriteRehearsalEligible: true` when the in-memory approval contract is satisfied. That value is not an execution path and does not perform or authorize a write by itself.

## Blocked Scope

The helper blocks requests that include:

- production DB write
- DB schema migration
- product import route connection
- API/UI import flow exposure
- package script exposure
- raw HRC access
- report JSON write
- copied DB path guard failure
- missing or invalid approval

## Privacy and Path Redaction

Output must not expose local/private path, user token, email, or raw HRC path strings. Approval tokens are intentionally not stored.

The helper returns redacted copied DB target path information from the path guard result and re-redacts any private token that reaches the output.

## Non-goals

- No DB copy.
- No copied DB creation.
- No import execution.
- No DB read.
- No DB write.
- No file write.
- No report JSON generation.
- No product import route connection.
- No API/UI import flow.
- No package script.
- No raw HRC zip access.
