# poker-tournament-lab v1.9 Plan

## v1.9 Summary

v1.9는 v1.8에서 저장 가능해진 `hand -> actions[]` multi-action v2 데이터를 Database Detail 안에서 더 쉽게 탐색하기 위한 Browser v2 foundation 버전입니다.

이번 버전은 새 solver가 아니라 read-only 화면 모델과 Database 중심 탐색 UI를 강화하는 작업입니다. 저장된 action별 frequency, size, EV, ChipEV, ICM EV, warning을 GTO Wizard식 Solution Browser에 가까운 형태로 볼 수 있게 만드는 기반을 제공합니다.

## UI Direction Comparison

### Option 1. Database 내부 Browser v2

- Existing Database list/detail flow를 유지합니다.
- 선택한 solution detail 안에 Browser v2 section을 추가합니다.
- v1.6 action/sizing summary, v1.7/v1.8 multi-action preview와 자연스럽게 연결됩니다.
- 신규 navigation surface 없이 점진적으로 확장할 수 있습니다.
- v1.9에서 선택한 방식입니다.

### Option 2. 별도 Solution Browser 탭

- 장기적으로 GTO Wizard식 탐색 경험을 독립적으로 만들기 좋습니다.
- matrix, filters, hand detail, EV mode를 더 넓게 배치할 수 있습니다.
- 다만 v1.9에서는 navigation surface와 상태 관리 범위가 커집니다.
- v2.0 이후 검토합니다.

### v1.9 Recommended UI Direction

v1.9에서는 Database Detail 안에 Browser v2 섹션을 추가하는 방식을 선택했습니다. 이 방식은 v1.8 stable의 multi-action storage/display 기반을 가장 작게 확장하고, v2.0에서 별도 Solution Browser 탭으로 분리할 수 있는 실험 기반을 제공합니다.
