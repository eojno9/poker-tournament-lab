## Goals

- Triage current npm audit warnings without running `npm audit fix`.
- Separate application blockers from non-blocking dependency maintenance items.
- Preserve package files until a dedicated dependency update step is approved.
- Keep public documentation neutral and avoid any data-origin or local-path disclosure.
- Preserve the off-table-only safety scope.

## Non-goals

- No dependency update.
- No `package.json` change.
- No `package-lock.json` change.
- No runtime, API, or UI code change.
- No database write.
- No raw data, raw archive, or generated report access.
- No GitHub push.
- No tag creation, deletion, or movement.
- No original repo remote addition.
- No live assistance, RTA, OCR, screen capture, overlay, hotkey, live watcher, or poker client integration.
