## v1.4 포함 기능
1. Trainer 문제 생성:
   - `buildTrainerProblemFromSolution(...)`
   - `gradeTrainerAnswer(...)`
2. Trainer 화면:
   - 문제 카드 + SHOVE/FOLD 제출 + 결과 카드
3. Trainer 기록:
   - recent (최대 30)
   - mistakes (최대 50, 오답만)
4. 문제 선택 옵션:
   - hero position / table size / tree config / source file
   - hand 직접 입력
   - seed 입력
5. 학습 요약:
   - 전체/최근 정답률, 최근 결과, 최근 오답, hand별 간단 집계
6. 테스트:
   - unit + smoke 시나리오 확장 완료
