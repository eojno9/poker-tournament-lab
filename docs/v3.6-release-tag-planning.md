## Release Summary

v3.6 improves the off-table Trainer experience without expanding the project into
live-play assistance or data persistence scope.

Included:

- Korean-first Trainer copy polish
- local session status and reset affordance
- clearer problem card and answer feedback
- mistake review status summary and empty states
- local summary/statistics readability improvements
- position/action local statistics headings
- safer localStorage fallback for Trainer history
- updated unit and smoke coverage

Excluded:

- server persistence
- account or cloud storage
- API route changes
- DB writes
- raw data handling
- generated report workflows
- new dependencies
- package script changes
- live/RTA/OCR/screen capture/overlay/hotkey/watcher/client integration
