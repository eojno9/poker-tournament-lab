# v3.0 Stable Release Decision

## Scope Freeze

v3.0 scope is frozen at the no-write product import foundation stage.

Included in scope:

- Product import preview contract and validation helpers.
- Existing solution snapshot adapter for in-memory duplicate/canonical key checks.
- Backup and rollback design.
- Backup manifest and copied-DB rehearsal manifest pure helpers.
- Copied DB path guard pure helper.
- No-write CLI command planning, output rendering, entrypoint design, and no-write entrypoint implementation.
- Controlled no-write manual invocation rehearsal.
- v3.0-rc2 privacy and backup policy hardening.

Not included in v3.0:

- Actual product DB import.
- Product import API or UI connection.
- `package.json` import/preview script exposure.
- DB schema migration.
- DB write.
- GitHub push.
