## 1) Goal
v1.1 focuses on **stabilization and UX improvements** so users can use the existing:
- exact HRC lookup (`HRC_PRECOMPUTED_DB`)
- fallback ICM path (`FALLBACK_ICM` / `NOT_SOLVED`)

No new solver capability is added in v1.1.

## 2) Guardrails
- Keep `v1.0-rc1` tag and backup untouched.
- Prefer UI/validation/usability over calculation changes.
- Do not change DB/API/fallback calculation logic unless absolutely required by UX wiring.
- Keep RTA/live features prohibited.

## 3) Explicit Non-Goals (v1.1)
- PKO / bounty / satellite special logic
- Nash approximation
- nearest match recommendation
- HRC raw export parser
- postflop solver / broad preflop tree solver
- hand history parser / trainer mode
- OCR / screen capture / overlay / hotkey / live watcher / real-time assistance
