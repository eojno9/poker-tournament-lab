# v3.0 Command Report Shape Helper

## Purpose

The command report shape helper defines the future dry-run import command report and summary shape as pure in-memory data.

This is not a command implementation. It does not add a `package.json` script, write report JSON, read or write the DB, call APIs, connect product import routes, or change UI/runtime behavior.

## Non-Goals

- Not an actual command.
- Not a `package.json` script.
- Not report JSON generation.
- Not file write.
- Not DB read.
- Not DB write.
- Not API/UI/runtime connection.
- Not product import route connection.
- Not schema migration.
- Not raw HRC artifact handling.

## Report Fields

The report shape includes:

- `version`
- `commandName`
- `mode`
- `timestampIso`
- `exitCode`
- `status`
- `previewSummary`
- `validationSummary`
- `backupManifestSummary`
- `safetyGateSummary`
- `dbSha256Before`
- `dbSha256After`
- `dbSha256Unchanged`
- `writeAllowed`
- `dbWriteAllowed`
- `reportFileWriteAllowed`
- `productImportRouteDisabled`
- `warnings`
- `nextAction`

`timestampIso` is provided by the caller. The helper does not call `Date.now`, generate ids, or inspect the filesystem.

## Exit Code Policy

Future command report exit codes:

- `0`: dry-run preview completed with no blocking safety issue.
- `1`: validation blocking issue.
- `2`: safety gate failure.
- `3`: privacy/path failure.
- `4`: unexpected internal or input shape error.

Priority:

1. privacy/path failure
2. safety failure
3. validation blocking issue
4. input shape invalid
5. success

## Write Policy

These fields are always false:

- `writeAllowed`
- `dbWriteAllowed`
- `reportFileWriteAllowed`

The helper cannot authorize DB writes or report file writes.

## DB SHA256 Policy

The report carries `dbSha256Before` and `dbSha256After` as caller-supplied values.

If they differ, the helper reports a safety failure with exit code `2`.

## Privacy / Path Failure Policy

If local path exposure, raw artifact exposure, failed privacy scan, or forbidden private tokens are detected, the report exits with code `3`.

The helper redacts public-facing strings containing:

- local Windows paths
- user tokens
- email addresses
- sample-external-hrc-folder fragments
- raw HRC folder labels
