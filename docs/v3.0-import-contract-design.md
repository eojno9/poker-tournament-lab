# v3.0 HRC Import Preview Contract Design

## Purpose

This document defines the v3.0 import preview contract that converts the v2.9 real HRC batch rehearsal classification into a product import preview shape.

The contract is intentionally review-only. It does not write to the database, connect product import routes, change schema, add API behavior, or add UI controls.

## Inputs From v2.9

v2.9 produced controlled dry-run and review outputs for 28 real HRC zip candidates:

- `IMPORT_CANDIDATE`: 19
- `NEEDS_MANUAL_REVIEW`: 8
- `HOLD`: 0
- `EXCLUDE`: 1

The input source documents are:

- `docs/v2.9-real-hrc-inventory-report.md`
- `docs/v2.9-controlled-batch-dry-run-report.md`
- `docs/v2.9-artifact-safety-dashboard-review.md`
- `docs/v2.9-import-candidate-classification.md`

## Explicit Non-Goals

- This is not a DB write implementation.
- This is not a DB schema migration.
- This does not connect `/api/imports/hrc` or any product import route.
- This does not add a UI import button.
- This does not read raw HRC zip files.
- This does not read external artifact JSON.
- This does not create or commit generated artifact JSON.
- This does not create or commit `artifacts/hrc-dry-run-reports`.

## Preview Row Contract

`HrcImportPreviewRow` is the review shape used before any import implementation exists.

Key fields:

- `id`
- `zipFileNameSanitized`
- `classification`
- `decision`
- `riskLevel`
- `reason`
- `nextAction`
- `dryRunSucceeded`
- `privacyPassed`
- `dashboardReviewed`
- `artifactReportAvailable`
- `importAllowed`
- `dbWriteAllowed`
- `sourceKind`
- `sourceVersion`
- `warnings`

`importAllowed` means only that a row may advance to import preview review. It does not permit a DB write.

`dbWriteAllowed` is always `false` in v3.0 Step 2.

## Decision Policy

| Input state | Decision | Risk | Import preview allowed | DB write allowed |
| --- | --- | --- | --- | --- |
| `IMPORT_CANDIDATE` + dry-run passed + privacy passed + dashboard reviewed | `READY_FOR_IMPORT_PREVIEW` | `LOW` | `true` | `false` |
| `NEEDS_MANUAL_REVIEW` | `MANUAL_REVIEW_REQUIRED` | `MEDIUM` | `false` | `false` |
| `HOLD` | `HOLD` | `HIGH` | `false` | `false` |
| `EXCLUDE` | `EXCLUDED` | `BLOCKED` | `false` | `false` |
| privacy failed | `MANUAL_REVIEW_REQUIRED` or `EXCLUDED` | `BLOCKED` | `false` | `false` |
| dashboard not reviewed | `MANUAL_REVIEW_REQUIRED` | `MEDIUM` | `false` | `false` |
| dry-run failed | `MANUAL_REVIEW_REQUIRED` or `EXCLUDED` | `MEDIUM` or `BLOCKED` | `false` | `false` |
