# v3.6 closeout

## Scope

v3.6 closes the local-only Trainer improvement bundle after the v3.5 Korean-first
Trainer release.

The implementation remains off-table only. It does not add server persistence, API
write paths, DB writes, raw data handling, generated report output, live assistance,
OCR, screen capture, overlays, hotkeys, watchers, or poker client integration.

payloads and attempts to reset corrupt payloads to an empty array. Storage write
failures are ignored so the Trainer remains usable when browser storage is full or
unavailable.
