The annotation states that the affected JavaScript actions target Node.js 20 and
are being forced to run on Node.js 24 by the GitHub Actions platform.

## Official Context

GitHub Actions is moving JavaScript action execution away from Node.js 20.
Node.js 20 reached end of life in April 2026, and GitHub-hosted runners are
transitioning affected JavaScript actions toward Node.js 24 execution.

The v3.4 workflow already requests Node.js 24 for project commands. The
annotation is about the runtime used by the GitHub-provided JavaScript actions,
not a failure in the project test/build/smoke commands.
